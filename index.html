CSV / Excel Paste Upload

Copy your schedule from Excel and paste below. Format: DEPT, NAME, 1, 2, 3, RD, 1, 2... (one row per employee, shifts separated by commas). Use RD for Rest Day. First column = Department, Second = Name, rest = shifts for days 1-31.

Manual Schedule Editor

Edit shifts directly in the table below. Click a cell and type 1, 2, 3, or RD. Use the month selector above.

How It Works

Priority: Firebase cloud data → Local browser data → Hardcoded fallback

1. Firebase: Saves to cloud for all team members to see. Requires Firestore write permissions.

2. Local: Saves only in this browser. Others won't see it.

3. Hardcoded: Built-in July & August 2026 schedules as fallback.

Excel to CSV Tip

In Excel, select your schedule table (including dept/name columns), copy, and paste directly into the text area above. The app will auto-detect the format.

You can also use tab-separated values - just paste from Excel and it will work.
